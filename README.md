# Pin-Fin-Experimental-Free-Convection-Laboratory
MATLAB and LaTeX workflow for analyzing convective heat transfer for various material pin fins in ambient air.

## Motivation
The goal of this project is to validate theory with experiments.

## 
Methods
- MATLAB scripts for convective coefficient and heat transfer calculation
- LaTeX documentation for process, derivations and results

## Results
Includes reproducible plots and comparisons with theoretical models, as well as analysis and explanations.

## Repository Structure
- `src/` : MATLAB scripts
- `docs/` : LaTeX reports
